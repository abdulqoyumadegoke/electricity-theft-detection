\# AI-Driven Electricity Theft Detection



This project explores electricity theft detection using machine learning and anomaly detection techniques.



\## Project Structure



\- Electricity.ipynb – Main notebook

\- electricity\_consumption\_dataset.xlsx – Dataset

\- processed/ – Processed data

\- models/ – Saved models

\- visualizations/ – Generated charts and figures
